#Hulu_clone
Hulu_clone is a clone of the Hulu system.

## 🚀 Techs

- ⚛️ Reactjs

## ✋🏻 Prerequisites

- [Node.js](https://nodejs.org/)

## 🔥 Install & Execute

1. Clone the repo;
2. Run `cd Hulu_Clone` to open the cloned repo;
3. Run `yarn install` to install the dependecies;
4. Run `ysrn start` to run the system;
